# Smart Healthcare Platform

A 4-member microservices-based smart healthcare and telemedicine platform.

## Tech Stack
- Frontend: React
- Backend: FastAPI
- Database: MongoDB
- AI: OpenAI API
- Video Consultation: Jitsi Meet
- Payments: Stripe Sandbox
- Notifications: Gmail SMTP
- Containerization: Docker
- Orchestration: Kubernetes

## Branching Strategy
- main -> stable branch
- develop -> integration branch
- feature branches -> individual member work branches

## Team Modules
- Member 1: Auth + Gateway + AI Symptom Checker
- Member 2: Patient Service + Notification Service + PDF Downloads
- Member 3: Doctor Service + Appointment Service + Status Tracker
- Member 4: Telemedicine Service + Payment Service
