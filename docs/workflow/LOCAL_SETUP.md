# Local Setup Checklist

Each team member must install the following:

## Required Tools
- VS Code
- Git
- Node.js LTS
- Python 3.11 or 3.12
- Docker Desktop
- Web Browser

## Recommended Tools
- MongoDB Compass
- Postman or Bruno
- VS Code Python Extension
- VS Code ESLint Extension
- VS Code Prettier Extension

## Notes
- MongoDB will be run using Docker later.
- Do not install random extra frameworks before the team setup is completed.
- All development should happen from the `develop` branch unless instructed otherwise.

## Version Check Commands

```powershell
git --version
node -v
npm -v
python --version
pip --version
docker --version
docker compose version
